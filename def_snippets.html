• Internet

    A network of computers around the world that share information with each other.

  • World Wide Web

    A collection of resources that can be accessed on the internet. We use HTTP requests (and responses) to access these.
    (webfoundation.org)

  • hypertext

    Text that contains a link to more text. (w3.org)

  • multimedia

    Different forms of expressing or communicating ideas.

  • CLI

    Command Line Interface

    A piece of software that uses just text (the command line) to interact with users.

  • GUI

    Graphical User Interface

    A piece of software that uses graphics on the screen to interact with users. This is the most used form of software today.

  • HCI

    Human. Computer. Interaction

    Designing software for other people.

  • FTP

    File Transfer Protocol

    A method for exchanging files over a network. (www.serv-u.com)

  • SSH

    Secure Socket Shell

    A method for securely accessing a computer on another network. (searchsecurity.techtarget.com)

  • HTTP

    Hyper Text Transfer Protocol

    A protocol that allows you to get resources from the web. (mozilla)

  • telnet

    A network protocol that lets a user on one computer log in to another computer that is on the same network. (kb.iu.edu)

  • HTML

    Hyper Text Markup Language

    The main markup language used to create web pages. (w3schools.com)

  • CSS

    Cascading Style Sheets

    Allows a developer to change the way pieces of elements look and act in HTML.
